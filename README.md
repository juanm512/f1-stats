# f1-stats
Formula 1 Season Summary App
This repository contains a mobile application built with React Native and Expo, which displays a summary of the current Formula 1 season.

Features
Displays a summary of the current Formula 1 season, including driver and constructor standings, race calendar, and results.
Provides detailed information about each race, such as track information, lap records, and weather conditions.
Allows users to filter the information by driver or team, and view race results for each driver or team.
Installation
To run the application locally, follow these steps:

Clone the repository to your local machine.
Install Expo CLI by running npm install -g expo-cli in your terminal.
Install project dependencies by running npm install in the project directory.
Run the application using expo start or npm start command.
Note: you may need to install Expo Go app on your mobile device to run the app on your phone.

Usage
Once you have the app running, you can use it to view the current Formula 1 season summary. You can navigate through different sections of the app using the bottom navigation bar, and use the filter option to view race results for specific drivers or teams.
